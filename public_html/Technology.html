HTML stands for Hyper Text Markup Language.
Hyper Text means text with links in it.
Markup Language means a language used to make text do more than sit on a page.
For example it can turn into an image,links, tables,lists,and much more.
HTML is mainly used to create web pages.

CSS is short for Cascading Style Sheets. CSS can be described as the
skin and make up that cover the bones of HTML.
In other words CSS makes web pages look pretty
